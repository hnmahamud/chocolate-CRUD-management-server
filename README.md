## API Reference

#### GET all chocolates

```http
  /chocolates
```

#### GET single chocolates

```http
  /chocolates/:id
```

#### POST single chocolates

```http
  /chocolates
```

#### PUT(Update) single chocolates

```http
  /chocolates/:id
```

#### DELETE single chocolates

```http
  /chocolates/:id
```

##Installation

For install dependency:

```sh
npm install
```

For run the project:

```sh
nodemon index.js
```

or

```sh
npm start
```
